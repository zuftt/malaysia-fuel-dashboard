🚀 API ENDPOINTS

Prices:
  GET  /api/v1/prices/latest              Current fuel prices
  GET  /api/v1/prices/history             Historical data (30+ days)
  GET  /api/v1/prices/compare             Malaysia vs global benchmarks
  GET  /api/v1/prices/volatility          Price volatility analysis

News:
  GET  /api/v1/news/latest                Recent announcements
  GET  /api/v1/news/alerts                Price change notifications
  GET  /api/v1/news/search                Keyword/tag filtering
  GET  /api/v1/news/by-source/{source}    Filter by source
  GET  /api/v1/news/tags                  All available tags

Trends:
  GET  /api/v1/trends/subsidy-gap         Subsidy cost analysis
  GET  /api/v1/trends/volatility          Price volatility metrics
  GET  /api/v1/trends/forecast            Price predictions
  GET  /api/v1/trends/comparison          Regional comparison

Admin:
  POST /api/v1/admin/prices/validate      Manual validation
  POST /api/v1/admin/announcements/manual Manual entry
  GET  /api/v1/admin/scraper-status       Scraper monitoring
  POST /api/v1/admin/alerts/config        Alert setup
  GET  /api/v1/admin/stats                Dashboard statistics

Auth:
  POST /api/v1/auth/login                 Get JWT token
  POST /api/v1/auth/register              Create account
  GET  /api/v1/auth/me                    Current user info
